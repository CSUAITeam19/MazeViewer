# MazeViewer
## 使用说明
- 基本操作
  - 左上角输入迷宫文件路径和搜索结果文件路径, 点击 "加载迷宫" 加载
  - 使用WASD和方向键控制移动, 鼠标拖动视角, 空格键上升, Shift键下降
  - 点击右下角的切换按钮切换到正交模式(平行俯视视角), WASD和方向键控制移动, 鼠标拖动画面, 滚轮缩放
  - 上方的进度条可以显示和控制搜索进度
  - 最外侧两个按钮分别是倒放和正放, 多次点击可加速
  - 次外层两个按钮分别是上一步和下一步
  - 在自动倒放或正放时, 中间的按钮可以停止自动播放
- 进程间通信模式
  - 迷宫编辑器会自动连接到本程序, 编辑器更新时触发本程序重新加载迷宫
